# 🏫 Campus Helper - 배재대학교 시간표 웹 애플리케이션

복잡한 학교 생활, 캠퍼스 헬퍼와 함께 더 스마트하게! 배재대학교 학생들을 위한 실시간 강의실 정보, 시간표 조회, 데이터 기반 편의 기능을 제공하는 올인원 웹 애플리케이션입니다.

**[🚀 Live Demo 바로가기](https://siwoo281.github.io/ClassLink/)**

---

## ✨ 주요 기능 (Key Features)

Campus Helper는 대학 생활의 편의성을 높이기 위한 다양한 데이터 기반 기능들을 제공합니다.

| 기능 | 설명 |
| --- | --- |
| **🏠 실시간 캠퍼스 현황** | 현재 시간 기준, 캠퍼스 내 모든 건물의 사용 중인 강의실과 빈 강의실 현황을 한눈에 파악할 수 있습니다. 건물별로 그룹화되어 있으며, 클릭 시 상세 정보를 확인할 수 있습니다. |
| **🔍 빈 강의실 검색** | 특정 요일과 시간을 지정하여 사용 가능한 모든 빈 강의실을 검색할 수 있습니다. 갑작스러운 팀 모임이나 공부할 장소가 필요할 때 유용합니다. |
| **🗓️ 통합 스케줄 조회** | 과목명, 교수명, 학과, 강의실 등 다양한 조건으로 전체 강의 스케줄을 검색하고 필터링할 수 있습니다. |
| **👨‍🏫 교수님별 상세 정보** | 교수님을 선택하면 해당 교수님의 주간 시간표, 총 강의 수/학점 등 통계 정보를 시각적으로 보여줍니다. 또한, 시간표의 빈칸을 통해 상담 가능 시간을 쉽게 추정할 수 있습니다. |
| **📊 캠퍼스 혼잡도** | 요일과 시간대별 캠퍼스 전체의 강의실 사용량을 히트맵(Heatmap)으로 시각화하여, 어느 시간대에 캠퍼스가 가장 붐비는지 직관적으로 파악할 수 있습니다. |
| **🍔 점심 룰렛** | 점심 메뉴 결정이 어려울 때, 간단한 룰렛을 돌려 메뉴를 추천받을 수 있는 재미 기능입니다. |
| **📱 반응형 UI** | 모바일, 태블릿, 데스크톱 등 모든 기기에서 최적화된 화면을 제공하여 언제 어디서든 편리하게 이용할 수 있습니다. |

## �️ 기술 스택 (Tech Stack)

-   **Frontend:** `HTML5`, `CSS3`, `JavaScript (ES6+)`
-   **Visualization:** `Chart.js` (with `chartjs-chart-matrix`)
-   **Data Processing:** `Python`
-   **Deployment:** `GitHub Pages`

## 📂 프로젝트 구조

```
/
├── index.html              # 메인 페이지
├── style.css               # UI 스타일 시트
├── script.js               # 프론트엔드 로직
├── timetable.json          # 웹앱에서 사용하는 최종 데이터
├── converter.py            # 원본 데이터를 timetable.json으로 변환하는 스크립트
├── 개설강좌 리스트.json    # 원본 데이터 파일
└── README.md               # 프로젝트 소개
```

## ⚙️ 로컬에서 실행하기 (Getting Started)

1.  **리포지토리 클론:**
    ```bash
    git clone https://github.com/siwoo281/ClassLink.git
    ```

2.  **디렉토리 이동:**
    ```bash
    cd ClassLink
    ```

3.  **로컬 서버 실행:**
    Python이 설치되어 있어야 합니다.
    ```bash
    python3 -m http.server 8000
    ```

4.  **브라우저에서 확인:**
    웹 브라우저를 열고 `http://localhost:8000` 주소로 접속합니다.

## � 데이터 처리 과정

이 프로젝트는 원본 데이터를 웹 애플리케이션에 맞게 가공하는 과정을 포함합니다.

1.  `개설강좌 리스트.json` 파일에 있는 원본 대학 강의 데이터를 `converter.py` 스크립트가 읽어들입니다.
2.  스크립트는 각 강의 정보를 분석하여 시간, 요일, 강의실 등을 분리하고 웹에서 사용하기 쉬운 구조로 정제합니다.
3.  최종적으로 `timetable.json` 파일이 생성되며, 웹 애플리케이션은 이 파일을 불러와 모든 기능을 동적으로 구현합니다.
